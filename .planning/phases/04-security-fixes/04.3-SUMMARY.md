# Phase 04.3 Summary — Dev-mode Supabase Warning

## One-liner
Added console.warn in supabase.ts when Supabase env vars are missing in dev mode.

## What was done
- Added `import.meta.env.DEV` guard in `src/lib/supabase.ts`
- When `VITE_SUPABASE_URL` or `VITE_SUPABASE_ANON_KEY` are missing, logs a clear warning to the console
- App continues to function with seed data fallback when Supabase is not configured

## Result
Developers starting the app without `.env.local` immediately see a console warning identifying the missing config.
